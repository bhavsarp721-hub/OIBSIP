# WebDev-L2-Calculator

## Project Description
This project is a simple and responsive Calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations through an easy-to-use interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear (C) button
- Delete (DEL) button
- Decimal number support
- Responsive design
- User-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure

```
OIBSIP/
└── WebDev-L1-Calculator/
    ├── index.html
    └── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the `WebDev-L1-Calculator` folder.
3. Double-click `index.html` to open the calculator in your web browser.

## Output

The calculator performs basic mathematical calculations such as addition, subtraction, multiplication, and division through a clean and responsive interface.
